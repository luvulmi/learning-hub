## FastAPI AI 추천 서비스 (`ai-service/`)

### uv 적용
- 기존 pip + venv 방식 → uv로 전환
- `pyproject.toml` + `uv.lock`으로 의존성 관리
- 실행: `uv run uvicorn main:app --reload --port 8000`

### Mock / 실제 AI 전환 구조
- `.env`의 `USE_MOCK=true/false` 환경변수 하나로 전환
- Mock 모드: 카테고리 기반 사전 정의 추천 반환 (API 키 없이 동작)
- 실제 AI 모드: Claude Haiku 호출 (API 키 발급 후 사용)

```
USE_MOCK=true             # 현재 모드 (Mock)
ANTHROPIC_API_KEY=...     # 발급 후 USE_MOCK=false로 전환
```

### API
- `GET /health` — 서비스 상태 + 현재 모드 확인
- `POST /recommend` — 학습 기록 목록 받아 다음 토픽 추천

### 추가된 파일
- `AiServiceClient` — RestClient로 FastAPI `/recommend` 호출
- `RecommendationService` — 최근 기록 10개 요약 → AI 호출 → DB 저장

## 버그 수정

### 401 시 OAuth 리다이렉트 문제
- 토큰 만료 시 Spring Security가 401 대신 구글 OAuth 플로우로 리다이렉트
- `authenticationEntryPoint` 추가 → 401 JSON 반환으로 수정
- axios 인터셉터가 401 감지 → `/login`으로 자동 이동

### `PageImpl` 직렬화 경고
- `@EnableSpringDataWebSupport(pageSerializationMode = VIA_DTO)` 추가
- 프론트 `Page<T>` 타입 구조도 `page.totalElements` 형태로 업데이트

### 상세 페이지 라우트 누락
- `/records/:id` 라우트가 없어서 카드 클릭 시 404
- `RecordDetailPage` 생성 및 라우터에 추가

## 프론트엔드 UI 개선

### 디자인 방향
- `slate` 계열 컬러 시스템으로 통일
- sticky 헤더 + blur 효과
- 전체 페이지 `max-w-4xl` + `px-6`으로 너비 통일 (헤더/본문 정렬)

### Button 컴포넌트 정리
- raw `<button>` 태그 → `<Button>` 컴포넌트로 전체 통일
- `active:scale-95` 눌리는 효과 추가
- `destructive` variant: 흰 배경 + 빨간 텍스트/테두리 (덜 공격적)

- `rounded-md` → `rounded-full` (pill 모양)
- `font-semibold` → `font-medium`
- hover 효과 제거

## 트러블슈팅

### GitHub 잔디(contribution) 미반영 문제
- **증상**: VSCode에서 커밋/푸시 해도 GitHub 잔디가 안 깔림
- **원인**: git 커밋 이메일(`k******@gmail.com`)이 GitHub 계정에 인증되지 않은 상태
- **GitHub 잔디 반영 조건**:
  - 커밋 이메일이 GitHub 계정에 등록 + 인증(Verified)된 이메일이어야 함
  - 기본 브랜치(main)에 push되거나 PR이 merge되어야 함
  - private 레포는 GitHub Settings → Contributions에서 "Private contributions" 활성화 필요
- **해결 방향**:
  - 방법 1: GitHub → Settings → Emails에서 `k******@gmail.com` 인증 (Resend verification email)
  - 방법 2: `git config --global user.email "인증된이메일"` 로 git 이메일 변경
- **VSCode는 무관**: IDE 종류는 잔디와 관계없음, git 이메일 설정이 핵심
